# Agri-Connect

A cloud-based web application that facilitates direct trade between farmers and consumers, eliminating intermediaries and ensuring fair prices.

## Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Appwrite, Supabase
- **Image Management**: Cloudinary
- **Email Service**: Nodemailer
- **UI Components**: Radix UI
- **Styling**: Lucide React Icons

## Getting Started

### Prerequisites

- Node.js and npm/pnpm installed

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm run dev
```

The application will be available at `http://localhost:3000`

### Build

```bash
pnpm run build
pnpm start
```

## Features

- Direct trade between farmers and consumers
- Real-time order tracking
- Secure payments and transactions
- Smart logistics and cluster-based distribution
- Image upload and product management
- User authentication and role-based access
